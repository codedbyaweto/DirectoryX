# DirectoryX

A modern **staff directory application** built with **React and TypeScript** that allows users to search, filter, and view detailed staff profiles in a clean and responsive interface.

The application demonstrates core React concepts including state management, conditional rendering, and dynamic filtering.

---

## Features

- Search staff members by **name or email**
- Filter staff by **department**
- Dynamic filtering that updates the UI instantly
- Clickable **staff cards** that open a detailed profile view
- Clean and responsive **grid-based layout**
- Simple **Back navigation** from detail view
- Built with **React + TypeScript**

---

## Tech Stack

- React
- TypeScript
- CSS Grid
- Vite
